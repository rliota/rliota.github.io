Projects

  • JSCompile

    JSCompile is a tool I wrote in Java to help manage large JavaScript projects, where sane development practices (object-oriented code, split into separate files) lead to insane namespace management issues.

    Originally only designed to support corralling files into namespaces and resolving their dependencies, the compiler has grown to leverage First-Class functions in javascript to simplify referencing of name-spaced objects.

    Similar to (and inspired by) Java package management, @import statements in comments instruct the compiler to enclose file contents in a function, and pass the name-space defined objects in as arguments named after their file name.
